# RAG Workflow

## Overview

This project is a simple RAG workflow using LangChain, Ollama Embeddings and Qdrant Vector Database.

## Setup

1. Start the Qdrant server

```console
docker compose up -d
```

2. Install the dependencies

```console
pip install -r requirements.txt
```

3. Run the workflow

```console
python src/main.py
```
